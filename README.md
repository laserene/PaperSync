# PaperSync

Automates the extraction of paper titles from specific Google Docs document and updates Google Sheets document

## Installation & Setup
Run the following command to install the necessary libraries.

```
python -m pip install -r requirements.txt
```

In order to extract the papers from Google Document and apply to Google Sheets, enter the document IDs to `config.py`. In addition, add the Groq API key to `.env`.
